# Humanoid Robot Favicon Skill

## Objective
Create or replace the website favicon with a unique, modern, and clean design that represents a humanoid robot.

---

## Instructions

1. **Design**
   - Represent a humanoid robot head or icon
   - Simplified, recognizable at small sizes (16x16, 32x32, 48x48)
   - Modern, flat or slightly 3D look
   - Minimal colors for clarity
   - Ensure it scales well as a small favicon

2. **Format**
   - Preferred formats: `.ico` or `.png`
   - Multiple sizes for proper browser support
     - 16x16, 32x32, 48x48, 64x64
   - Optional: 180x180 for Apple touch icon

3. **Color Palette**
   - Futuristic, tech-inspired colors (blue, silver, gray)
   - High contrast for visibility
   - Avoid too many details; focus on recognizability

4. **Integration**
   - Replace the current favicon in Docusaurus: `/static/img/favicon.ico` or `/static/img/favicon.png`
   - Update `docusaurus.config.js` if needed:
     ```js
     favicon: 'img/favicon.ico',
     ```
   - Ensure it works on all modern browsers

5. **Optional Enhancements**
   - Slight shadow or glow effect for depth
   - Minimalistic robotic facial features (eyes, antennas, lines)
   - Maintain a professional look for documentation

---

## Output
1. Generated favicon file(s) in `.ico` or `.png`
2. Instructions to replace the existing favicon in the Docusaurus project
3. Optional preview of how it looks at different sizes
