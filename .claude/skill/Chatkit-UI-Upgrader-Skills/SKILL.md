# ChatKit UI Modernization Skill

## Objective
Enhance the visual appearance, responsiveness, and user experience of the ChatKit component integrated into the book.

---

## Instructions

1. **Chat Window Layout**
   - Modern, clean, slightly rounded container
   - Drop shadow or subtle border for separation from page
   - Match book’s upgraded color palette

2. **Message Bubbles**
   - Differentiate user and agent messages with colors
   - Rounded corners, subtle shadows
   - Smooth appearing animation for new messages

3. **Input Area**
   - Modern input box with rounded corners
   - Send button with hover/press animation
   - Responsive sizing for mobile and desktop

4. **Header/Footer**
   - Optional: Add subtle header with agent name/logo
   - Footer could include status or typing indicator

5. **Colors & Typography**
   - Use same fonts and theme as upgraded book
   - High readability for all message types
   - Ensure good contrast and accessible colors

6. **Responsiveness**
   - Mobile: chat window expands full width
   - Desktop: fixed-width sidebar or floating chat
   - Ensure touch-friendly buttons

7. **Integration**
   - Do NOT change chat logic, API calls, or backend
   - Do NOT break existing RAG functionality
   - Styling only

---

## Output
1. CSS and optional component overrides for ChatKit UI
2. Responsive design adjustments
3. Hover/animation effects for messages and buttons
4. Instructions to integrate upgraded ChatKit UI in Docusaurus
