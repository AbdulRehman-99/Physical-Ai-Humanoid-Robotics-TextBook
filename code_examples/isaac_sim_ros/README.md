# Isaac Sim ROS Examples

This directory contains code examples and project setups for integrating humanoid robots with NVIDIA Isaac Sim and ROS 2 for perception and navigation tasks.

## Setup

1.  **Install Isaac Sim**: Follow the official NVIDIA Isaac Sim documentation for installation.
2.  **ROS 2 Bridge**: Ensure the ROS 2 Bridge is configured within Isaac Sim.
3.  **Project Structure**:
    -   `run_slam.py`: Example script for SLAM.
    -   `run_navigation.py`: Example script for autonomous navigation.
    -   `edge_cases.py`: Example script for handling noisy sensor data.

## Usage

Refer to Chapter 3 of the book for detailed instructions on how to set up and run these examples.
